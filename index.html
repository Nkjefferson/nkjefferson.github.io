Projects

Hive Boardgame Simulator

Created a move simulator for the table top board game Hive using Object-Oriented Analysis and Design concepts. Through multiple itterations I implemented all move restrictions and an infinite board size for every version. Also included the creation of an AI that did not preform a non-legal move based on the current version of the game in play, ranging 5 different versions of the game.

Cypher Gather Word Generator

Designed a random word generator to be used by a freestyle rap club that allowed for crowd sourcing words through the use of the Twilio api. This allowed the audience to text in suggested words in real time that they wanted to see used in the show. Deployed the application using a Flask back end on Heroku.

Random Quote Generator

Developed a random quote generator that uses json to pull quotes from a third party website. Using javascript and sccs I created a dynamically updating page that shifts color and quote displayed on the screen. This application also has the functionality to directly tweet the quote for the user.

Embedded Assembley Line Sorting Arm

Using an AVR Atmel 664 Microprocessor, I worked on a team to develop the drivers for various sensors, using i2c and UART protocols. The robot used C on an embedded processor to calculate the difference between heavy and light weights, and sorted them while synchronizing with the conveyor belt that was passing the weights infront of it.

Room Mapping And Navigation Robot

Developing and testing the software platform using robot operating system to navigate unknown locations. Processing maps to create explored and frontier regions to navigate to a location that would allow for additional information to be determined. Obstacle avoidance and advanced A-star path planning were also implemented.

Agile Task Manager Module

Collaborated on a team of eleven members to create an Agile workflow task manager application. This application included features such as workflow breakdown, colorblind mode, urgency tracking, and real-time updating. Through working on a pre-existing codebase we implemented a completely new feature to the janeway program that functions similarly to other task manager applications such as Trello.
